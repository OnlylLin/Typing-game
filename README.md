## 功能特色

### 11种游戏模式

| 模式 | 图标 | 描述 |
|------|:----:|------|
| 经典模式 | ⚔️ | 输入目标句子，实时高亮反馈，测速与测准 |
| 词库练习 | 📚 | tech/code/quote/science/文学/中文/自定义词库 |
| 计时挑战 | ⏱️ | 30/60/120秒限时挑战，完成尽可能多的句子 |
| 极速挑战 | ⚡ | 限时内尽量多打，只计正确字符 |
| 幽灵竞速 | 👻 | 对抗历史最佳净WPM，实时进度条对比 |
| 每日一挑 | 📅 | 每天同一题，首次完成奖励 x1.5 |
| 盲打模式 | 🦅 | 记忆后隐藏目标，凭记忆完成 |
| 渐进挑战 | 🏰 | 9关由易到难，限时递减，支持暂停 |
| 段落挑战 | 📖 | 更长段落文本，考验持久耐力 |
| Boss关卡 | 👹 | 30关Boss战斗，回合制打字对战 |
| 弱键练习 | 💪 | 自动分析错误，生成针对性训练 |

### Boss战斗系统

- **11个独特Boss**：从史莱姆王到深渊领主，难度递进
- **30关挑战**：Boss关、小怪关、选择题关交替
- **9种玩家技能**：回血、护盾、暴击、冰冻、解毒、重击、持续恢复、净化、闪避
- **Boss技能**：毒害、灼烧、冰冻、眩晕、恐惧、完形填空攻击
- **弱点系统**：速度弱点、精准弱点、连击弱点等，命中造成双倍伤害
- **状态效果系统**：中毒扣血、灼烧伤害、冰冻无法行动等

### 词汇学习系统

- **CET4/CET6词汇库**：含中文释义和词形变化
- **选择题系统**：英译中、中译英、形态选择
- **90+道完形填空**：含语法解析和知识点说明
- **错误追踪**：记录错误字符、位置、类型分布
- **针对性练习建议**：基于错误数据生成练习建议

### 进度与成长

- **等级系统**：积累经验升级，解锁称号
- **技能系统**：消耗经验升级6种被动技能
- **成就系统**：40+成就，分类展示
- **每日任务**：9项任务每天刷新
- **详细统计**：WPM趋势图、错误热力图、模式分布

### 实时反馈

- **实时打字**：无需按Enter，即时高亮反馈
- **连击系统**：连续正确输入显示连击数
- **幽灵进度条**：玩家与幽灵并排进度对比
- **暂停功能**：段落/渐进挑战支持暂停
- **错误闪烁**：输入错误时背景闪红提示

## 快速开始

### 环境要求

- Python 3.8 或更高版本

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/OnlylLin/Typing-game.git
cd Typing-game

# 运行游戏
python game.py
```

### 首次运行

游戏会自动创建 `stats.json` 和 `config.json` 文件保存进度和设置。

## 操作指南

### 通用快捷键

| 按键 | 功能 |
|------|------|
| `ESC` | 退出当前模式 |
| `P` | 暂停（段落/渐进挑战） |
| `Backspace` | 删除字符 |
| `Enter` | 确认输入 |

### Boss战斗

1. **攻击**：输入显示的单词拼写
2. **释义选择**：拼写正确后选择正确中文释义（全额伤害 vs 减半伤害）
3. **技能**：输入 `1-9` 使用对应技能
4. **策略**：合理使用技能，注意Boss弱点和状态效果

### 选择题

- 输入 `A/B/C/D` 选择答案
- 输入 `E` 表示不知道（惩罚减半，进入学习模式）

## 技术特点

- **跨平台支持**：Windows、macOS、Linux
- **暗黑RPG风格UI**：自定义颜色主题，Unicode边框
