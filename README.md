## 功能特色

### 游戏模式
| 模式 | 描述 |
|------|------|
| 经典模式 | 输入目标句子，实时高亮反馈，测速与测准 |
| 极速挑战 | 时限内尽量多打，只计正确字符 |
| 词库练习 | tech/code/quote/science/文学/随机/中文/自定义 |
| 计时挑战 | 自选 30/60/120 秒，完成尽量多 |
| 盲打模式 | 记忆后隐藏，凭记忆完成 |
| 渐进挑战 | 9关由易到难，限时递减，全通胜利 |
| 弱键练习 | 自动分析错误，生成针对性训练 |
| 每日一挑 | 每天同一题，首次完成奖励 x1.5 |
| 幽灵竞速 | 对抗历史最佳净WPM，实时进度对比 |
| 段落挑战 | 更长段落文本，考验持久耐力 |
| Boss关卡 | 30关Boss战斗，回合制打字对战 |

### Boss战斗系统
- 11个独特Boss，从史莱姆王到深渊领主
- 玩家技能：回血、护盾、暴击、冰冻、解毒
- Boss技能：毒害、灼烧、冰冻、眩晕、恐惧、完形填空攻击
- 状态效果系统和技能冷却机制

### 词汇学习系统
- CET4/CET6词汇库，含中文释义和词形变化
- 选择题系统：英译中、中译英、形态选择
- 完形填空练习，含语法解析
- 错误追踪与针对性练习建议

### 其他功能
- 等级系统与经验值
- 每日任务和成就系统
- 详细的统计数据和历史趋势图
- 错误热力图分析
- 可视化庆祝特效
- 实时错误追踪与改正率统计
- 针对性练习建议

## 快速开始

### 环境要求
- Python 3.8 或更高版本

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/yourusername/typing-game.git
cd typing-game

# 运行游戏
python game.py
```

### 配置（可选）
游戏支持以下设置：
- 实时打字模式（推荐）/ 传统输入模式
- 游戏提示开关
- 默认难度选择
- 自定义词库

## 操作指南

### 通用快捷键
| 按键 | 功能 |
|------|------|
| `ESC` | 退出当前模式 |
| `P` | 暂停（段落/渐进挑战） |
| `Backspace` | 删除字符 |

### Boss战斗
- 输入单词拼写 → 选择正确释义 → 造成伤害
- 输入 `1-5` 使用技能
- 答对释义：全额伤害
- 答错/不知道：伤害减半
